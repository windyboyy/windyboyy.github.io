## 5. Third-Party Services and International Data Transfers

This application uses the following third-party network services for IP geolocation. When using these services, your IP address may be transferred outside your country:

### 5.1 NetEase IP Query Service (mail.163.com)
- **Provider**: NetEase, Inc. (China)
- **Function**: Obtain current device's public IP address and geolocation
- **Data Sent**: Network request (automatically carries public IP)
- **Data Returned**: Public IP address, country, province, city, ISP

### 5.2 Bilibili IP Query Service (api.live.bilibili.com)
- **Provider**: Bilibili Inc. (China)
- **Function**: Query geographic location of specified IP
- **Data Sent**: IP address to query
- **Data Returned**: Country, province, city, ISP, coordinates

### 5.3 ipinfo.io ASN Query Service (api.ipinfo.io)
- **Provider**: IPinfo Inc. (USA)
- **Function**: Query AS number (Autonomous System Number) of IP
- **Data Sent**: IP address to query
- **Data Returned**: AS number, AS name, country
- **International Transfer Safeguard**: This service complies with GDPR requirements

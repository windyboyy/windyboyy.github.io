## 2. Informações que Coletamos e Base Legal

**Base Legal (Artigo 6 do RGPD):**
- **Execução de Contrato**: Necessário para fornecer os serviços de diagnóstico de rede que você solicita
- **Interesses Legítimos**: Para melhorar a experiência do aplicativo (realizamos uma avaliação de equilíbrio)

### 2.1 Informações do Dispositivo (lidas localmente para exibição)
- Nome do dispositivo, modelo do dispositivo

### 2.3 Dados de Uso (armazenados localmente)
- Histórico de destinos Ping
- Histórico de consultas DNS
- Histórico de detecção de portas TCP/UDP
- Histórico de Traceroute
- Histórico de requisições HTTP
- Histórico de testes de conexão
- Histórico de diagnóstico rápido
- Histórico de consultas IP

### 2.4 Identificador de Usuário (gerado localmente)
- ID de convidado (gerado aleatoriamente, usado apenas para identificação local)

## 4. Armazenamento e Segurança das Informações

1. **Local de Armazenamento**: Localmente no seu dispositivo (UserDefaults)
2. **Período de Armazenamento**: Até você limpar o histórico ou desinstalar o aplicativo
3. Não carregaremos nenhum dado para servidores
4. **Medidas de Segurança**: Os dados são armazenados apenas no seu dispositivo, protegidos pelo sandbox do sistema iOS

## 5. Serviços de Terceiros e Transferências Internacionais de Dados

Este aplicativo usa os seguintes serviços de rede de terceiros para geolocalização IP. Ao usar esses serviços, seu endereço IP pode ser transferido para fora do seu país:

### 5.1 Serviço de Consulta IP NetEase (mail.163.com)
- **Provedor**: NetEase, Inc. (China)
- **Função**: Obter endereço IP público e geolocalização do dispositivo atual
- **Dados Enviados**: Requisição de rede (carrega automaticamente o IP público)
- **Dados Retornados**: Endereço IP público, país, província, cidade, ISP

### 5.2 Serviço de Consulta IP Bilibili (api.live.bilibili.com)
- **Provedor**: Bilibili Inc. (China)
- **Função**: Consultar localização geográfica do IP especificado
- **Dados Enviados**: Endereço IP a consultar
- **Dados Retornados**: País, província, cidade, ISP, coordenadas

### 5.3 Serviço de Consulta ASN ipinfo.io (api.ipinfo.io)
- **Provedor**: IPinfo Inc. (EUA)
- **Função**: Consultar número AS (Autonomous System Number) do IP
- **Dados Enviados**: Endereço IP a consultar
- **Dados Retornados**: Número AS, nome AS, país
- **Proteção de Transferência Internacional**: Este serviço está em conformidade com os requisitos do RGPD

De acordo com RGPD, CCPA e outras leis aplicáveis, você tem os seguintes direitos:

1. **Direito de Acesso**: Você tem o direito de obter uma cópia dos dados pessoais que mantemos sobre você
2. **Direito de Retificação**: Você tem o direito de solicitar a correção de dados imprecisos
3. **Direito ao Apagamento (Direito de ser Esquecido)**: Você pode limpar todo o histórico no aplicativo ou excluir todos os dados desinstalando
4. **Direito à Limitação do Processamento**: Você tem o direito de solicitar limitação do processamento de dados
5. **Direito à Portabilidade de Dados**: Você tem o direito de obter seus dados em formato estruturado
6. **Direito de Objeção**: Você tem o direito de se opor ao processamento baseado em interesses legítimos
7. **Direito de Retirar o Consentimento**: Você pode retirar o consentimento a qualquer momento
8. **Direito de Reclamar**: Você tem o direito de apresentar uma reclamação a uma autoridade de proteção de dados

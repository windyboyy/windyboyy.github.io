# 개인정보 처리방침 요약

**최종 업데이트: 2026년 1월 5일**

이 요약은 개인정보를 어떻게 수집하고 사용하는지 빠르게 이해하는 데 도움이 됩니다.

> **중요**: 이것은 순수한 로컬 애플리케이션입니다. 모든 데이터는 장치에 로컬로 저장되며 서버에 업로드되지 않습니다.

## 수집하는 정보 (로컬에만 저장)

- **장치 정보**: 장치 모델, 운영 체제 버전, 장치 식별자 (로컬 표시용)
- **네트워크 정보**: 로컬 IP 주소, 네트워크 유형 (WiFi/셀룰러)
- **사용 데이터**: 탐색 대상 기록, IP 쿼리 기록
- **사용자 식별자**: 로컬에서 생성된 게스트 ID

## 법적 근거 (GDPR 제6조)

- **계약 이행**: 요청하신 네트워크 진단 서비스 제공에 필요
- **정당한 이익**: 애플리케이션 경험 및 기능 개선

## 정보 사용 방법

- 네트워크 탐색 서비스 제공
- 장치 네트워크 환경 정보 표시
- 편리한 향후 사용을 위한 기록 저장

## 네트워크 요청 알림

- IP 위치 기능은 지리적 위치 정보를 얻기 위해 공용 IP 주소를 타사 서비스로 전송합니다
- 이 외에는 개인정보를 서버로 전송하지 않습니다

## 귀하의 권리 (GDPR 및 관련 법률에 따라)

- **접근권**: 수집한 데이터 보기
- **삭제권**: 언제든지 기록을 지우거나 앱을 제거하여 모든 데이터 삭제
- **처리 제한권**: 데이터 사용 방법 제한
- **데이터 이동권**: 일반 형식으로 데이터 내보내기
- **동의 철회권**: 언제든지 동의 철회

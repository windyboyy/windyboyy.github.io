# 타사 SDK 목록

**최종 업데이트: 2026년 1월 5일**

이 애플리케이션은 다음 SDK를 통합합니다:

## 1. Apple 시스템 SDK

### 1.1 Foundation
- **제공자**: Apple Inc.
- **기능**: 기본 프레임워크 지원, 기본 데이터 유형 및 컬렉션 제공
- **수집 정보**: 없음

### 1.2 SwiftUI
- **제공자**: Apple Inc.
- **기능**: 사용자 인터페이스 프레임워크, 앱 인터페이스 구축
- **수집 정보**: 없음

### 1.3 UIKit
- **제공자**: Apple Inc.
- **기능**: UI 컴포넌트 지원, 장치 정보 획득
- **수집 정보**: 장치 이름, 시스템 버전, 배터리 상태

### 1.4 Network
- **제공자**: Apple Inc.
- **기능**: 네트워크 통신 지원, TCP/UDP 연결, 네트워크 상태 모니터링
- **수집 정보**: 네트워크 연결 상태

### 1.5 CoreTelephony
- **제공자**: Apple Inc.
- **기능**: 셀룰러 네트워크 정보 획득
- **수집 정보**: 셀룰러 네트워크 유형 (2G/3G/4G/5G)

### 1.6 NetworkExtension
- **제공자**: Apple Inc.
- **기능**: 네트워크 확장 지원 (패킷 캡처 기능용 예약)
- **수집 정보**: 없음

### 1.7 Security
- **제공자**: Apple Inc.
- **기능**: 보안 프레임워크 지원
- **수집 정보**: 없음

### 1.8 Darwin
- **제공자**: Apple Inc.
- **기능**: 저수준 시스템 호출, Ping, Traceroute 등에 사용
- **수집 정보**: 없음

### 1.9 dnssd
- **제공자**: Apple Inc.
- **기능**: DNS 서비스 검색, DNS 쿼리 기능에 사용
- **수집 정보**: 없음

## 2. 타사 상용 SDK

이 애플리케이션은 현재 다음을 포함한 타사 상용 SDK를 통합하지 않습니다:
- 광고 SDK 없음
- 분석 SDK 없음
- 소셜 공유 SDK 없음
- 푸시 알림 SDK 없음
- 결제 SDK 없음

## 3. 타사 네트워크 서비스

### 3.1 NetEase IP 쿼리 서비스
- **서비스 URL**: mail.163.com
- **제공자**: NetEase, Inc.
- **기능**: 현재 장치의 공용 IP 주소 및 위치 획득
- **전송 데이터**: 네트워크 요청 (공용 IP 자동 전송)
- **반환 데이터**: 공용 IP 주소, 국가, 지역, 도시, ISP

### 3.2 Bilibili IP 쿼리 서비스
- **서비스 URL**: api.live.bilibili.com
- **제공자**: Bilibili Inc.
- **기능**: 지정된 IP의 지리적 위치 쿼리
- **전송 데이터**: 쿼리할 IP 주소
- **반환 데이터**: 국가, 지역, 도시, ISP, 좌표

### 3.3 ipinfo.io ASN 쿼리 서비스
- **서비스 URL**: api.ipinfo.io
- **제공자**: IPinfo Inc. (미국)
- **기능**: IP의 AS 번호 (자율 시스템 번호) 쿼리
- **전송 데이터**: 쿼리할 IP 주소
- **반환 데이터**: AS 번호, AS 이름, 국가

## 4. 참고

1. 이 애플리케이션은 안전성과 신뢰성을 보장하기 위해 Apple 공식 시스템 SDK만 사용합니다.
2. 향후 새로운 타사 SDK가 통합되면 이 목록을 업데이트하고 즉시 통지합니다.
3. 모든 SDK 사용은 최소 필요 원칙을 따릅니다.

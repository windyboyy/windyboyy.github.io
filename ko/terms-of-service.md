# 서비스 이용약관

**최종 업데이트: 2026년 1월 5일**
**시행일: 2026년 1월 5일**

NetPulse에 오신 것을 환영합니다!

## 1. 서비스 설명

NetPulse(이하 "서비스")는 다음을 포함하는 네트워크 품질 모니터링 도구입니다:
- 로컬 네트워크 진단 도구 (Ping, Traceroute, DNS 쿼리, TCP/UDP 포트 감지 등)
- HTTP 요청 테스트
- 네트워크 연결 테스트
- 빠른 진단 서비스
- 속도 테스트
- 지연 시간 테스트
- IP 위치 조회
- 장치 정보 보기

## 2. 사용자 등록 및 계정

1. 게스트 로그인을 통해 서비스를 이용할 수 있으며, 시스템이 자동으로 고유 사용자 식별자를 할당합니다(로컬에 저장).
2. 장치를 안전하게 보관해야 합니다. 장치의 무단 사용으로 인한 손실에 대해 서비스는 책임지지 않습니다.

## 3. 사용자 행동 지침

서비스 사용 시 다음 지침을 준수해야 합니다:
1. 해당 관할권의 법률 및 규정 준수
2. 불법 활동에 서비스를 사용하지 않음
3. 다른 사람의 네트워크를 악의적으로 탐색하거나 공격하지 않음
4. 서비스의 정상적인 운영을 방해하지 않음
5. 네트워크 스캔, 포트 스캔 또는 타인의 권리를 침해할 수 있는 활동에 서비스를 사용하지 않음

## 4. 서비스 변경 및 종료

1. 비즈니스 필요에 따라 서비스의 일부 또는 전체를 변경, 중단 또는 종료할 권리가 있습니다.
2. 본 약관을 위반하는 경우, 서비스 제공을 종료할 권리가 있습니다.
3. 앱 내 알림을 통해 서비스 변경 또는 종료를 사전에 통지합니다.

## 5. 면책 조항

1. 서비스에서 제공하는 네트워크 탐색 결과는 참고용이며 어떠한 형태의 보증도 구성하지 않습니다.
2. 불가항력으로 인한 서비스 중단에 대해 책임지지 않습니다.
3. 속도 테스트 결과는 다양한 요인의 영향을 받으며 참고용입니다.
4. IP 위치 정보는 타사 서비스에서 제공되며 오류가 포함될 수 있습니다.

## 6. 데이터 설명

1. 이 애플리케이션의 모든 데이터는 로컬 장치에 저장됩니다. 개인 데이터를 수집하거나 업로드하지 않습니다.
2. 앱 내에서 언제든지 기록을 삭제할 수 있습니다.
3. 앱을 제거하면 모든 로컬 데이터가 삭제됩니다.
4. 데이터 처리에 대한 자세한 내용은 개인정보 처리방침을 참조하세요.

## 7. 지적 재산권

1. 이 애플리케이션의 모든 콘텐츠(인터페이스 디자인, 아이콘, 텍스트 포함)는 지적 재산권 법률의 보호를 받습니다.
2. 허가 없이 이 애플리케이션의 어떤 부분도 복사, 수정 또는 배포할 수 없습니다.

## 8. 준거법 및 분쟁 해결

1. 본 약관의 해석, 유효성 및 분쟁 해결은 해당 관할권의 법률에 따릅니다.
2. 분쟁은 먼저 우호적인 협상을 통해 해결해야 합니다.

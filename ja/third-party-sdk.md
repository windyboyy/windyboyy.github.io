# サードパーティSDKリスト

**更新日：2026年1月5日**

本アプリケーションは以下のSDKを統合しています：

## 1. AppleシステムSDK

### 1.1 Foundation
- **提供者**：Apple Inc.
- **機能**：基本フレームワークサポート、基本データ型とコレクションを提供
- **収集情報**：なし

### 1.2 SwiftUI
- **提供者**：Apple Inc.
- **機能**：ユーザーインターフェースフレームワーク、アプリインターフェースを構築
- **収集情報**：なし

### 1.3 UIKit
- **提供者**：Apple Inc.
- **機能**：UIコンポーネントサポート、デバイス情報を取得
- **収集情報**：デバイス名、システムバージョン、バッテリー状態

### 1.4 Network
- **提供者**：Apple Inc.
- **機能**：ネットワーク通信サポート、TCP/UDP接続、ネットワーク状態監視
- **収集情報**：ネットワーク接続状態

### 1.5 CoreTelephony
- **提供者**：Apple Inc.
- **機能**：モバイルネットワーク情報を取得
- **収集情報**：モバイルネットワークタイプ（2G/3G/4G/5G）

### 1.6 NetworkExtension
- **提供者**：Apple Inc.
- **機能**：ネットワーク拡張サポート（パケットキャプチャ機能用に予約）
- **収集情報**：なし

### 1.7 Security
- **提供者**：Apple Inc.
- **機能**：セキュリティフレームワークサポート
- **収集情報**：なし

### 1.8 Darwin
- **提供者**：Apple Inc.
- **機能**：低レベルシステムコール、Ping、Tracerouteなどに使用
- **収集情報**：なし

### 1.9 dnssd
- **提供者**：Apple Inc.
- **機能**：DNSサービス検出、DNSクエリ機能に使用
- **収集情報**：なし

## 2. サードパーティ商用SDK

本アプリケーションは現在、以下を含むサードパーティ商用SDKを統合していません：
- 広告SDKなし
- 分析SDKなし
- ソーシャル共有SDKなし
- プッシュ通知SDKなし
- 支払いSDKなし

## 3. サードパーティネットワークサービス

### 3.1 NetEase IPクエリサービス
- **サービスURL**：mail.163.com
- **提供者**：NetEase, Inc.
- **機能**：現在のデバイスのパブリックIPアドレスとジオロケーションを取得
- **送信データ**：ネットワークリクエスト（パブリックIPを自動的に送信）
- **返却データ**：パブリックIPアドレス、国、省、市、ISP

### 3.2 Bilibili IPクエリサービス
- **サービスURL**：api.live.bilibili.com
- **提供者**：Bilibili Inc.
- **機能**：指定IPの地理位置をクエリ
- **送信データ**：クエリするIPアドレス
- **返却データ**：国、省、市、ISP、座標

### 3.3 ipinfo.io ASNクエリサービス
- **サービスURL**：api.ipinfo.io
- **提供者**：IPinfo Inc.（米国）
- **機能**：IPのAS番号（自律システム番号）をクエリ
- **送信データ**：クエリするIPアドレス
- **返却データ**：AS番号、AS名、国

## 4. 注意事項

1. 本アプリケーションは安全性と信頼性を確保するため、Appleの公式システムSDKのみを使用しています。
2. 将来新しいサードパーティSDKが統合される場合、このリストを更新し、速やかに通知します。
3. すべてのSDKの使用は最小必要原則に従います。

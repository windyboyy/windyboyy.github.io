4. **Mesures de sécurité** : Les données sont stockées uniquement sur votre appareil, protégées par le sandbox du système iOS

## 5. Services tiers et transferts internationaux de données

Cette application utilise les services réseau tiers suivants pour la géolocalisation IP. Lors de l'utilisation de ces services, votre adresse IP peut être transférée en dehors de votre pays :

### 5.1 Service de requête IP NetEase (mail.163.com)
- **Fournisseur** : NetEase, Inc. (Chine)
- **Fonction** : Obtenir l'adresse IP publique et la géolocalisation de l'appareil actuel
- **Données envoyées** : Requête réseau (transporte automatiquement l'IP publique)
- **Données retournées** : Adresse IP publique, pays, province, ville, FAI

### 5.2 Service de requête IP Bilibili (api.live.bilibili.com)
- **Fournisseur** : Bilibili Inc. (Chine)
- **Fonction** : Interroger la localisation géographique de l'IP spécifiée
- **Données envoyées** : Adresse IP à interroger
- **Données retournées** : Pays, province, ville, FAI, coordonnées

### 5.3 Service de requête ASN ipinfo.io (api.ipinfo.io)
- **Fournisseur** : IPinfo Inc. (États-Unis)
- **Fonction** : Interroger le numéro AS (Autonomous System Number) de l'IP
- **Données envoyées** : Adresse IP à interroger
- **Données retournées** : Numéro AS, nom AS, pays
- **Protection des transferts internationaux** : Ce service est conforme aux exigences du RGPD

## 2. Informasi yang Kami Kumpulkan dan Dasar Hukum

**Dasar Hukum (GDPR Pasal 6):**
- **Pelaksanaan Kontrak**: Diperlukan untuk menyediakan layanan diagnostik jaringan yang Anda minta
- **Kepentingan yang Sah**: Untuk meningkatkan pengalaman aplikasi (kami telah melakukan penilaian keseimbangan)

### 2.1 Informasi Perangkat (dibaca secara lokal untuk tampilan)
- Nama perangkat, model perangkat
- Versi sistem operasi
- Pengidentifikasi perangkat (pengidentifikasi model, bukan IDFA/IDFV)
- Level baterai, status baterai
- Ruang disk, penggunaan memori

### 2.2 Informasi Jaringan (dibaca secara lokal untuk diagnostik jaringan)
- Alamat IP lokal (IPv4/IPv6)
- Jenis jaringan (WiFi/2G/3G/4G/5G)
- Status koneksi jaringan

### 2.3 Data Penggunaan (disimpan secara lokal)
- Riwayat target Ping
- Riwayat kueri DNS
- Riwayat deteksi port TCP/UDP
- Riwayat Traceroute
- Riwayat permintaan HTTP
- Riwayat uji koneksi
- Riwayat diagnosis cepat
- Riwayat kueri IP

### 2.4 Pengidentifikasi Pengguna (dibuat secara lokal)
- ID Tamu (dibuat secara acak, hanya digunakan untuk identifikasi lokal)

## 5. Layanan Pihak Ketiga dan Transfer Data Internasional

Aplikasi ini menggunakan layanan jaringan pihak ketiga berikut untuk geolokasi IP. Saat menggunakan layanan ini, alamat IP Anda mungkin ditransfer ke luar negeri:

### 5.1 Layanan Kueri IP NetEase (mail.163.com)
- **Penyedia**: NetEase, Inc. (Tiongkok)
- **Fungsi**: Mendapatkan alamat IP publik dan geolokasi perangkat saat ini
- **Data Terkirim**: Permintaan jaringan (secara otomatis membawa IP publik)
- **Data Dikembalikan**: Alamat IP publik, negara, provinsi, kota, ISP

### 5.2 Layanan Kueri IP Bilibili (api.live.bilibili.com)
- **Penyedia**: Bilibili Inc. (Tiongkok)
- **Fungsi**: Mengkueri lokasi geografis IP yang ditentukan
- **Data Terkirim**: Alamat IP untuk dikueri
- **Data Dikembalikan**: Negara, provinsi, kota, ISP, koordinat

### 5.3 Layanan Kueri ASN ipinfo.io (api.ipinfo.io)
- **Penyedia**: IPinfo Inc. (AS)
- **Fungsi**: Mengkueri nomor AS (Nomor Sistem Otonom) IP
- **Data Terkirim**: Alamat IP untuk dikueri
- **Data Dikembalikan**: Nomor AS, nama AS, negara
- **Perlindungan Transfer Internasional**: Layanan ini mematuhi persyaratan GDPR

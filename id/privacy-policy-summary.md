# Ringkasan Kebijakan Privasi

**Terakhir Diperbarui: 5 Januari 2026**

Ringkasan ini dimaksudkan untuk membantu Anda dengan cepat memahami bagaimana kami mengumpulkan dan menggunakan informasi pribadi Anda.

> **PENTING**: Ini adalah aplikasi lokal murni. Semua data disimpan secara lokal di perangkat Anda dan tidak akan diunggah ke server kami.

## Informasi yang Kami Kumpulkan (hanya disimpan secara lokal)

- **Informasi Perangkat**: Model perangkat, versi sistem operasi, pengidentifikasi perangkat (untuk tampilan lokal)
- **Informasi Jaringan**: Alamat IP lokal, jenis jaringan (WiFi/Seluler)
- **Data Penggunaan**: Riwayat target probe, riwayat kueri IP
- **Pengidentifikasi Pengguna**: ID tamu yang dibuat secara lokal

## Dasar Hukum (GDPR Pasal 6)

- **Pelaksanaan Kontrak**: Diperlukan untuk menyediakan layanan diagnostik jaringan yang Anda minta
- **Kepentingan yang Sah**: Untuk meningkatkan pengalaman dan fungsionalitas aplikasi

## Bagaimana Kami Menggunakan Informasi

- Menyediakan layanan probe jaringan
- Menampilkan informasi lingkungan jaringan perangkat
- Menyimpan riwayat untuk penggunaan yang nyaman di masa depan

## Pemberitahuan Permintaan Jaringan

- Fitur lokasi IP mengirimkan alamat IP publik Anda ke layanan pihak ketiga untuk mendapatkan informasi lokasi geografis
- Selain itu, kami tidak mengirimkan informasi pribadi Anda ke server mana pun

## Hak Anda (berdasarkan GDPR dan hukum yang berlaku)

- **Hak Akses**: Melihat data yang kami kumpulkan tentang Anda
- **Hak Penghapusan**: Menghapus riwayat kapan saja atau menghapus semua data dengan menghapus instalasi aplikasi
- **Hak Pembatasan Pemrosesan**: Membatasi cara kami menggunakan data Anda
- **Hak Portabilitas Data**: Mengekspor data Anda dalam format umum
- **Hak Penarikan Persetujuan**: Menarik persetujuan Anda kapan saja
